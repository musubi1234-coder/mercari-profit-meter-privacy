プライバシーポリシー

最終更新日: 2026年3月21日

メルカリ利益メーター（以下「本拡張機能」）は、ユーザーのプライバシーを最優先に設計されています。本ポリシーでは、本拡張機能におけるデータの取り扱いについて説明します。

1. 収集するデータ

本拡張機能は、個人情報を一切収集しません。

  • 氏名、メールアドレス、住所等の個人識別情報は収集しません
  • 閲覧履歴やアクセスログは収集しません
  • メルカリのアカウント情報にはアクセスしません
  • ユーザーの入力した仕入原価や梱包費の情報は、ユーザーのブラウザ内（chrome.storage.sync）にのみ保存され、外部サーバーには一切送信しません

2. データの保存

本拡張機能は、以下のデータをユーザーのブラウザ内にのみ保存します：

  • デフォルトの配送方法設定
  • デフォルトの梱包費
  • 振込手数料の表示設定
  • パネルのサイズ設定

これらはすべてユーザーの利便性向上のためであり、chrome.storage.sync APIを使用してユーザーのChromeアカウントに紐づく端末間で同期されます。外部サーバーへの通信は行いません。

3. 外部通信

本拡張機能は外部サーバーとの通信を一切行いません。すべての計算処理はユーザーのブラウザ内でローカルに実行されます。アナリティクスツール、広告SDK、トラッキングピクセル等は含まれていません。

4. 第三者へのデータ共有

本拡張機能はデータを収集しないため、第三者へのデータ共有、販売、提供は一切ありません。

5. メルカリのページへのアクセス

本拡張機能は jp.mercari.com の商品ページにアクセスし、以下の情報をページ上から読み取ります：

  • 商品の販売価格
  • 配送方法の種類と送料

これらの情報はリアルタイムの利益計算にのみ使用され、保存・記録・送信されることはありません。

6. Google API の利用について

本拡張機能が Google API から取得する情報の利用は、Chrome Web Store User Data Policy（Limited Use 要件を含む）に準拠します。

7. 権限（パーミッション）の利用目的

  • storage: ユーザーの設定（配送方法、梱包費等）をブラウザに保存するため
  • activeTab: ユーザーが閲覧中のメルカリ商品ページの価格を読み取るため
  • host_permissions (jp.mercari.com): メルカリ商品ページに利益計算パネルを表示するため

8. お子様のプライバシー

本拡張機能は13歳未満のお子様を対象としておらず、意図的にお子様から個人情報を収集することはありません。

9. ポリシーの変更

本ポリシーは必要に応じて更新されることがあります。重要な変更がある場合は、拡張機能の更新を通じてお知らせします。

10. お問い合わせ

本拡張機能のプライバシーに関するご質問は、Chrome Web Store の拡張機能ページからお問い合わせください。

Privacy Policy (English)

Last updated: March 21, 2026

Mercari Profit Meter ("the Extension") is designed with user privacy as our top priority.

Data Collection: The Extension does not collect any personal information. No browsing history, account information, or personally identifiable information is collected, stored, or transmitted to external servers.

Local Storage: User preferences (default shipping method, packaging cost, etc.) are stored locally in the user's browser using chrome.storage.sync and are never sent to external servers.

No External Communication: The Extension performs all calculations locally within the browser. No analytics tools, advertising SDKs, or tracking pixels are included.

Google API Compliance: The Extension's use of information received from Google APIs adheres to the Chrome Web Store User Data Policy, including the Limited Use requirements.

Contact: For privacy-related inquiries, please contact us through the Extension's Chrome Web Store page.
